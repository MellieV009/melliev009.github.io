Emiel spul

Console into the router and enable privileged EXEC mode:
switch> enable

Go into the config mode:
switch#configure terminal
Enter configuration commands, one per line. End with CNTL/Z.
switch(config)#

Changing the hostname of a switch to VoorbeeldNaam:
switch(config)#hostname VoorbeeldNaam
VoorbeeldNaam(config)#

Set the routers domain name as voorbeeld.com:
router(config)# ip domain name voorbeeld.com

To add a banner message (motd):
router(config)#banner motd $ Authorized Users Only! $

Exit config mode:
switch(config)#exit

Enable password is used for securing privilege mode:
switch(config)#enable secret VOORBEELD

Enable password for Line console. Set login and set logout after four minutes (not required):
switch(config)#line console 0
switch(config-line)#password VOORBEELD
switch(config-line)#exec-timeout 4 0
switch(config-line)#login

Encrypt the plaintext passwords:
switch(config)# service password-encryption

Configure the system to require a minimum 12-character password:
switch(config)# security passwords min-length 12

Configure the username SSHadmin with an encrypted password of TWAALFTEKENSWACHTWOORD:
switch(config)# username SSHadmin secret TWAALFTEKENSWACHTWOORD

Generate a set of crypto keys with a 1024 bit modulus:
switch(config)# crypto key generate rsa general-keys modulus 1024

Copy to startup-configuration file from running-configuration file:
switch#copy running-config startup-config

To watch startup-configuration file and running-configuration file:
switch#show startup-config
switch#show running-config

Create a vlan and give it a name:
vlan 10
name VOORBEELD

Assign ports to VLANs:
interface f0/1
switchport mode access
switchport access vlan 10

Configure the default gateway on switch:
switch(config)#ip default-gateway 1.1.1.1

Assign VLANs on switch based on the VLAN and Port Assignments. Ports are in access mode:
switch(config)#interface range f0/11 - 17
switch(config-if-range)#switchport mode access
switch(config-if-range)#switchport access vlan 10

Configure G0/1 of switch as a static trunk and assign the native VLAN:
switch(config)#interface g0/1
switch(config-if)#switchport mode trunk
switch(config-if)#switchport trunk native vlan 10

Disable all ports that are not assigned to a VLAN:
S1(config)#interface range f0/1-5,g0/2
S1(config-if-range)#shutdown

Configure router to exclude the first 10 addresses from the other routers LANs. All other addresses should be available in the DHCP address pool:
router(config)#ip dhcp excluded-address 192.168.10.1 192.168.10.10

Create a DHCP pool named VOORBEELD. Configure the DHCP pool to include the network address, the default gateway, and the IP address of the DNS server:
router(config)#ip dhcp pool VOORBEELD
router(dhcp-config)#network 192.168.10.0 255.255.255.0
router(dhcp-config)#default-router 192.168.10.1
router(dhcp-config)#dns-server 192.168.20.254

Configure a router as a DHCP relay agent:
router(config)#interface g0/0
router(config-if)#ip helper-address 1.1.1.1

Configure router as DHCP client:
router(config)#interface g0/1
router(config-if)#ip address dhcp
router(config-if)#no shutdown

  • Je moet LLDP gaan gebruiken om devices binnen het netwerk te vinden, KIJK DIT!
